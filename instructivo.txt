Entorno de Conda:

Crear un entorno de conda con python 3.11 -> "conda create -n "myenv" python=3.11"

## ATENCION ##
Si ya existe una carpeta build creada, BORRARLA
##############

Se deben instalar los siguientes paquetes
	conda install -c conda-forge boost
	conda install -c conda-forge flatbuffers

El proyecto se encuentra en el repositorio de gitlab, ejecutar "git clone https://gitlab.fing.edu.uy/nanoraw/pod5_nanoraw.git"

(git checkout 'mybranch')

Generar los archivos _version.py:
	echo "# file generated by setuptools_scm
	# don't change, don't track in version control
	__version__ = version = '0.2.4'
	__version_tuple__ = version_tuple = (0, 2, 4)" > _version.py


modificar el script build.sh: 
	En la linea 53 "conda activate pod5_nanoraw" -> "conda activate myenv"
	Linea 145: borrar el "pre-commint uninstall"


dentro de src/python/pgnano/constants modificar el archivo constants.py:
	Linea 1 : project_root="/data/pinanoraw/tgonzalez/" #Guardar ruta del proyecto
	Linea 7 : venv_interpreter_path = "/data/pinanoraw/tgonzalez/.conda/envs/pgnano/bin/python3.11" # Guardar ruta del entorno conda

La primera vez ejecutar que se quiere compilar el programa, ejecutar el script build.sh con el parametro init:
	"./build.sh init"
Posteriormente 
	"./build.sh c <clean|dirty> <release|debug|profile>"

Para comprimir/descomprimir un archivo se utiliza el programa copy:
	"./copy <archivo_entrada> <archivo_salida> <--uncompressed | --VBZ | --pgnano>"
